# NAILA ECOMMERCE FLUTTER

## Bahasa Indonesia

Naila E-Commerce adalah aplikasi e-commerce mobile yang dikembangkan menggunakan Flutter dan Dart. Aplikasi ini menyediakan fitur marketplace untuk membantu pengguna mencari, memilih, dan membeli produk secara digital.

### Fitur

- Autentikasi pengguna, meliputi login, pendaftaran akun, dan lupa kata sandi
- Marketplace dengan kategori, detail produk, favorit, dan keranjang belanja
- Checkout, pelacakan pesanan, serta pengelolaan profil pengguna

### Teknologi

Flutter, Dart, Provider State Management

### Cara Menjalankan

Jalankan `flutter pub get`, selanjutnya jalankan `flutter run` pada terminal.

---

## English

Naila E-Commerce is a mobile e-commerce application developed using Flutter and Dart. This application provides marketplace features that allow users to browse, select, and purchase products digitally.

### Features

- User authentication, including login, account registration, and password recovery
- Marketplace with categories, product details, favorites, and a shopping cart
- Checkout, order tracking, and user profile management

### Technologies

Flutter, Dart, Provider State Management

### How to Run

Run `flutter pub get`, then run `flutter run` in the terminal.
