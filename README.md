# NAILA E-COMMERCE

Project aplikasi e-commerce berbasis mobile menggunakan Flutter dengan fitur marketplace interaktif.

## Fitur

* Antarmuka autentikasi (login, daftar akun, dan lupa password)
* Halaman marketplace dengan kategori dan tampilan produk
* Detail produk, favorit, dan keranjang belanja
* Sistem checkout dan pelacakan pesanan
* Manajemen profil dan pengaturan aplikasi
* Kustomisasi tema dan tampilan responsif

## Teknologi

* Flutter
* Dart
* Provider State Management

## Cara Menjalankan

```bash
flutter pub get
flutter run
```
